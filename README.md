# ntest

A simple and lightweight library for testing in C++20.

Table of contents:
- [Features](#features)

## Features

- No 3rd party dependencies, just the STL
- No complicated build system, just 2 files
- Simple but useful reports
- Built-in assertion functions for:
  - integral types:
    - `int8_t`, `uint8_t`
    - `int16_t`, `uint16_t`
    - `int32_t`, `uint32_t`
    - `int64_t`, `uint64_t`
  - containers:
    - `char*` strings
    - `std::string`
    - `T[]`
    - `std::vector`
    - `std::array`
  - text files
